# Murphi-dt4
Murphi-dt4是协议验证软件paraVerifier服务器部分的变种。与paraVerifie服务器不同的是，Murphi-dt4使用决策树机制筛选paraVerifie客户端发来的候选不变式。相较于原版paraVerifier服务器，Murphi-dt4能够使用更少的内存判断候选不变式的正确性，从而减少协议验证对内存的要求。
